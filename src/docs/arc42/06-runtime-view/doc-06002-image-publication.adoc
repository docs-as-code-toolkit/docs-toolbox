== Image Publication Runtime Scenario

[plantuml, docs-toolbox-publication, svg]
----
@startuml
actor Maintainer
participant "GitHub Actions" as Actions
participant "Docker Buildx" as Buildx
database "GHCR" as Ghcr

Maintainer -> Actions : push to main, push tag, or workflow_dispatch
Actions -> Actions : compute image name and tags
Actions -> Actions : extract README description
Actions -> Buildx : build linux/amd64 and linux/arm64
Buildx -> Ghcr : push tags and cache
Actions --> Maintainer : print hash tag and publication summary
@enduml
----
